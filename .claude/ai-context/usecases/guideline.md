| ID | 행위자 | 내용 | 가치/이유 |
|----|-------|------|---------|

## UC-008: 번역 사전 조회 (View Translation Dictionary)

**행위자(Actor)**: 번역버디, 관리자

**관련 사용자 스토리**: US-017

**전제 조건(Preconditions)**:
- 사용자가 카카오톡 챗봇과 대화 중 (또는 관리자의 경우 웹 인터페이스에 유효한 인증 토큰으로 접근)
- 시스템이 모집 페이즈 이후 상태

**기본 흐름(Main Flow)**:
1. 사용자가 사전 조회 요청
2. 시스템이 사전 항목 목록 표시
	- 한국어 용어
	- 영어 대응어
3. [검색 기능] 사용자가 키워드 입력 시 필터링 결과 표시

**종료 조건(Postconditions)**:
- 사전 항목이 표시됨

## UC-009: 번역 사전 수정 (Edit Translation Dictionary)

**행위자(Actor)**: 관리자

**관련 사용자 스토리**: US-018

**전제 조건(Preconditions)**:
- 관리자가 웹 인터페이스에 유효한 인증 토큰으로 접근

**기본 흐름(Main Flow)**:
1. 관리자가 사전 항목 목록 조회
2. 관리자가 새 항목 추가, 수정, 삭제 선택 (자유롭게 편집 가능)

**새 항목 추가 흐름**:
3a. 관리자가 한국어 용어 입력 (최대 200자)
4a. 관리자가 영어 대응어 입력 (최대 200자)
5a. 시스템이 새 항목 저장

**수정 흐름**:
3b. 관리자가 기존 항목 선택
4b. 관리자가 내용 수정
5b. 시스템이 항목 업데이트

**삭제 흐름**:
3c. 관리자가 기존 항목 선택
4c. 시스템이 항목 삭제

**종료 조건(Postconditions)**:
- 추가: 새 사전 항목이 저장됨
- 수정: 사전 항목이 업데이트됨
- 삭제: 사전 항목이 삭제됨

## UC-010: 사전 자동 매핑 (Auto-map Dictionary Terms)

**행위자(Actor)**: 번역버디, 관리자

**관련 사용자 스토리**: US-019

**전제 조건(Preconditions)**:
- 시스템이 모집 페이즈 이후 상태
- 사전에 항목이 존재

**기본 흐름(Main Flow)**:
1. 사용자가 원본 텍스트 붙여넣기
2. 시스템이 사전 항목과 텍스트 매칭 수행
3. 시스템이 매핑된 용어만 필터링하여 표시
	- 원본 한국어 용어
	- 대응 영어 용어
	- 텍스트 내에서의 매핑 위치 (선택)

**종료 조건(Postconditions)**:
- 사전에 매핑된 치환 가능 단어 목록이 표시됨
